# OpenAPI 3.2.0 Compliance Report

This project aims for full compliance with the OpenAPI 3.2.0 Specification.
Currently, it successfully parses, emits, and merges elements including:
- OpenAPI Document Structure
- Paths and Operations
- Schema Objects (DTOs)
- Parameters (Path, Query)
- Request Bodies
- Responses

Ongoing efforts:
- Full Webhooks resolution
- Advanced Reference Resolution (100% test coverage ongoing)
- Callbacks implementation

Compliance Check is implemented in the test suite under the `Oas32Compliance` test coverage classes.
